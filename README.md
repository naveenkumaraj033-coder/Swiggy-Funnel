# 📊 Swiggy Funnel Analysis Dashboard

## 🚀 Project Overview
This project analyzes Swiggy’s end-to-end customer funnel to identify key factors impacting user behavior, conversion rates, and order volumes. The dashboard provides a comprehensive view of funnel performance, pricing impact, and operational metrics to support data-driven decision-making.

---

## 🎯 Objective
- Analyze customer journey across funnel stages  
- Identify drop-offs and conversion bottlenecks  
- Understand impact of pricing (discounts, delivery, packaging)  
- Provide actionable insights to improve conversion and performance  

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)  
- Excel (Data Cleaning & Preparation)  
- Data Analysis & Statistical Techniques  

---

## 📊 Dashboard Highlights

### 🔹 Key KPIs
- **28.22M** – Average Listings  
- **394.91K** – Average Restaurants  
- **340.49** – Payment Success Rate  
- **19.31** – Overall Conversion Rate  

---

### 🔹 Visual Insights
- 📉 **Funnel Analysis** – Tracks drop-offs across customer journey  
- 🎯 **Average Discount** – Measures impact of discounts on conversions  
- 💰 **Cost for Two Orders** – Trend of pricing across months  
- 📦 **Packaging vs Delivery Costs** – Cost comparison influencing orders  
- 🛒 **Cart/Menu Analysis** – User interaction and engagement trends  
- 💳 **Orders vs Payments** – Payment success distribution  
- 📈 **Menu vs Listings** – Platform availability vs engagement  

---

## 🔍 Key Insights
- Significant drop-offs observed in **Cart to Payment stages**, indicating conversion bottlenecks  
- **Higher delivery and packaging costs** negatively impact order completion  
- **Discounts play a critical role** in improving user conversion rates  
- **Menu availability vs listings mismatch** affects user engagement  
- Monthly trends show **fluctuations in pricing directly impacting order volumes**  

---

## 🚀 Achievements / Impact
- Identified key funnel bottlenecks impacting conversion rates  
- Provided insights to optimize **pricing and discount strategies**  
- Improved understanding of **user behavior and engagement patterns**  
- Built an interactive dashboard to monitor KPIs and performance

## 📈 Features
